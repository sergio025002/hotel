Reservas de quarto:

  • O sistema deve permitir que os clientes pesquisem a disponibilidade de quartos por datas específicas.
  • Os clientes devem poder selecionar o tipo de quarto desejado (individual, duplo, suíte, etc.).
  • O sistema deve permitir que os hóspedes façam reservas de quartos para as datas desejadas.

Gestão de reservas

  • Os funcionários do hotel devem poder visualizar as reservas existentes.
  • Os funcionários devem poder confirmar, alterar ou cancelar reservas de acordo com as normas e políticas do hotel.
  • O sistema deve enviar notificações por e-mail/celular dos clientes para confirmar suas reservas.

Pagamento

  • O sistema deve permitir que os clientes efetuem o pagamento das reservas online.
  • Deve ser possível visualizar o histórico de pagamentos e transações financeiras.
